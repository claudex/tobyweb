**tobyweb** is the frontend to display collected and computed datas from *tobybur*, the ruby bot to watch tribunes.

Requires
========

* `Django`_ version 1.3.x;
* `autobreadcrumbs`_;
* Django debug toolbar

And the database from *tobybur*.
